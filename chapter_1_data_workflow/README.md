# Chapter 1 - Data-Driven Workflow

This chapter documents the data collection, cleaning, API interaction, visualisation, and machine learning workflow for the project **From Inside Looking Out**.

## Datasets
- Airbnb reviews
- Airbnb interior images
- film clips and extracted frames
- Dutch paintings

## Goals
- collect and clean project-related data
- annotate images and texts
- visualise patterns in interior-outward looking scenes
- prepare embeddings for SOM analysis
