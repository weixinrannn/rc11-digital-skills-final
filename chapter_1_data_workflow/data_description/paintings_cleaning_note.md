# Paintings Cleaning Note

The painting dataset was cleaned in three stages:

1. exact duplicate removal
2. perceptual similarity removal
3. thematic filtering for interior-window scenes

## Selection Criteria
The final selected paintings were expected to show:
- an interior space
- a visible window
- a readable inside-outside relationship
- clear light logic
- the window as an important visual element

## Removal Criteria
The removed images generally included:
- pure outdoor landscapes
- paintings without windows
- portraits unrelated to window scenes
- interior scenes without an outward-looking relationship
- blurry or low-value images

## Result Summary
- raw images: 1452
- after exact duplicate cleaning: 928
- after similar-image cleaning: 915
- final selected images: 586
- removed / review images: 866
