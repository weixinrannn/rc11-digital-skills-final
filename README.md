# RC11 25117914

Student ID: 25117914

This project explores a data-driven design workflow using three datasets:
paintings, film frames, and Airbnb interior images.

## Chapter 1 – Data Workflow
Paintings are processed using YOLO and SAM to extract window-related fragments.

## Chapter 2 – Projecting Between Domains
CLIP and SOM are used to project different datasets into a shared semantic space.

## Chapter 3 – Spatial Generation
Data is translated into spatial systems using Blender.
